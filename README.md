# 🔍 Forensic Evidence Management System

## 📌 Overview

The **Forensic Evidence Management System** is a web-based DBMS project developed using **Flask** and **MySQL**. It helps investigators manage criminal cases, evidence records, investigators, and chain of custody efficiently.

---

## 🚀 Features

- Case Management
- Investigator Management
- Evidence Management
- Custody Management
- Dashboard
- CRUD Operations
- Search Functionality
- Responsive User Interface

---

## 🛠 Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS
- Google Colab
- ngrok

---

## 📂 Database Tables

- Cases
- Investigators
- Evidence
- Custody
